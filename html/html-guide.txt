Table in html
----Structure of Table
----------------
1 . method
    =>Table
        ->caption
        =>tr (table row)
            ->th (table heading)
            ->td (table data)
-----------------
2 . => table
    =>thead - it cover upper portion of table
        ->tr
        ->th
    =>tbody - it cover middle portion of table
        ->tr
        ->td
    =>tfoot - it cover bottom portion of table
        ->tr
        ->th
------------------
List of property in html table tr, th, td
    ->align
    ->border
    ->cellpadding - it apply space inside th & td
    ->cellspacing - it apply space outside of th & td
    ->width - th & td
    ->height - table, tr, th, td
    ->bordercolor
    ->bgcolor
    ->colspan - it use to combine space of 2 or more th & td
    ->rowspan - it use to combine space of 2 or more tr
-------------------

List of input tags
    ->text
    ->number
    ->password
    ->email
    ->file
    ->date
    ->datetime-local
    ->checkbox
    ->radio
    ->textarea
    ->select
    ->button
        =>button
        =>submit
        =>reset

----------------------

List of property of input tags
    ->placeholder - it is for input guidance.
    ->maxlength - it use to input limit in - text, email, password, textarea.
    ->size - it use to reset the width of - text, email, pwd.
    ->rows - it use to set height of textarea
    ->cols - it use to set width of textarea
    ->value - it use to display default value in - text, email, password, number, checkbox, radio.
    ->readonly - it will make input element readonly.
    ->disabled - it disabled / block the input tags.
