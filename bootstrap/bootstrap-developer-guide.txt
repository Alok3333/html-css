bootstrap is library of css.
it has collections of pre-defined css classes.
it use to create responsive webpage without writing any css code.
the bootstrap in came market in 2011 and it was introduced by twitter.

The bootstrap designed based on grid system.
The gris system has devided all type of devices in 6 type based on screen width.


1. extra small      - 1px to 576px screen width
2. small device     - 577px to 768px screen width
3. medium device    - 769px to 992px screen width
4. large device     - 993px to 1200px screen width
5. extra large device       - 1201px to 1400px screen width
6. extra extra large device - 1401px to extra large device

    while Creating webpage using bootstrap, follow these steps
        1. link to bootstrap.css file in head section
        2. follow body tag.
        3. inside body tag, follow these classes
            .container - it has margin from left and right
            .container-fluid - it has no margin like it cover full width of screen
        4. inside container of container-fluid, follow these classes
            .row - height is auto and width 100% container or container-fluid

        5. inside .row class, follow columns.
            =>in each row min 1 column and max 12 column only
